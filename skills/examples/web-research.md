name: web-research
triggers: research, look up, search for, find out, what is, who is
description: Research a topic using available tools
---

When asked to research something:

1. Use available browser tools (Chrome, Puppeteer) to search the web
2. Visit 2-3 relevant sources, not just the first result
3. Cross-reference information between sources
4. Summarize findings in a clear, structured way:
   - Key facts
   - Sources used
   - Confidence level (well-documented vs. sparse info)
5. Flag if information might be outdated or uncertain

If no browser tools are available, say so and suggest the owner search manually.
