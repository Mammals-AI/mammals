Claude Code
Your AI brain — powers everything

Claude Code is the AI engine that powers Mammals. You need a Claude Pro or Max subscription from Anthropic.

If you don't have it installed yet:

  1. Go to Claude Code install page
  2. Follow the install instructions for Mac
  3. Open Terminal and run: claude login
  4. Come back here and click "Check Again"

What is Terminal? It's a built-in app on every Mac that lets you type commands. Think of it like texting your computer — you type a command, press Enter, and it does something.

How to open it:

  • • Press Cmd + Space to open Spotlight
  • • Type Terminal and press Enter
  • • Or click here to open it automatically

How to run a command: When Terminal opens, you'll see a blinking cursor. Just type (or paste) the command and press Enter. That's it.

How to paste: Press Cmd + V — same as any other app.

Don't worry — you won't break anything. Terminal only does what you tell it to.
